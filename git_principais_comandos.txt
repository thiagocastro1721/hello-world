git principais comandos

Sua Identidade

	git config --global user.name "Fulano de Tal"
	git config --global user.email fulanodetal@exemplo.br

Seu Editor

	$ git config --global core.editor vim

Testando Suas Configurações

	git config --list
	
	exemplo:
	user.name=Fulano de Tal
	user.email=fulanodetal@exemplo.br
	color.status=auto
	color.branch=auto
	color.interactive=auto
	color.diff=auto
	
Inicializando um Repositório num Diretório Existente

	para o Linux:
	$ cd /home/user/my_project
	
	para o Mac:
	$ cd /Users/user/my_project
	
	para o Windows:
	$ cd /c/user/my_project
	
	e digita:
	$ git init

Clonar um Repositório Existente

	git clone <url>
	exemplo
	git clone https://github.com/libgit2/libgit2
	
Ao iniciar escrita no git commit
	Insert

Ao terminar a escrita do comit
	esc

Sair do git commit ou do git log
	shift + z + z

Preparar um diretório
	git init

Sair do git log
	q

Historico de mudancas
	git log

Empurrar para nuvem
	git push

Remover manualmente index.lock ou usar o comando
	rm -f .git/index.lock

SALVAR E SAIR DO VIM
	CTRL + Z + Z

Mudar branch

	git checkout <branch>

	exemplo

	git checkout calculadora_de_datas

Adicionar todos os aquivos

	git add --all

	Se aparecer
	Everything up-to-date

	está tudo ok.
	
Ver status e ver a branch atual

	git status
